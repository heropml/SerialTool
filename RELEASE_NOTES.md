# SerialTool v1.0.1

iOS 风格串口调试工具，本次带来数据区交互、批量发送、关键字高亮三大块增强，并补齐高分屏适配。

## ✨ 新功能

**数据区**

- **滚动锁定 / 自动跟随** —— 往上翻看历史时视图自动定住，数据照常接收不被打断；右下角浮出「↓ 最新」按钮，点一下回到底部并恢复自动跟随。
- **单击行高亮** —— 鼠标点数据区任意一行即整行高亮，再点取消，方便盯住某一帧。
- **关键字高亮** —— 标题栏「关键字高亮」按钮打开配置弹窗：可配多条关键字，每条独立颜色，可选「背景」或「文字」着色，并能限定「收 / 发 / 收发」范围；区分大小写匹配、自动跳过时间戳、规则持久化。
- **只显高亮行** —— 一个开关，数据区只保留命中关键字的行，其余折叠隐藏（数据不丢，关掉即恢复）。

**发送区**

- **多条发送** —— 「多条发送」按钮打开弹窗：每行一条数据 + 复选框，**每行可独立设 HEX / 换行 / 校验**；勾选多条后「循环发送」按设定间隔依次轮发；条目持久化。

**界面**

- **高分屏（HiDPI）缩放适配** —— 在 2K/4K 等高分辨率屏幕上界面与字体不再过小。
- **弹窗深色标题栏** —— 多条发送 / 关键字高亮弹窗的原生标题栏跟随主题深浅。
- **切主题不再"看不见字"** —— 切换主题时数据区已有文字按角色（时间戳 / 收 / 发）重涂成新主题色，浅↔深来回切都清晰可见。

## 🔧 改进

- 波特率下拉补全常用高速率：**256000 / 500000 / 512000 / 600000 / 750000 / 1000000 / 1500000 / 2000000**（仍可手填自定义）。
- 英文界面串口设置标签宽度自适应，不再被输入框遮挡。
- 数据区显示格式（HEX / 文本）现在**统一只由「HEX 显示」开关决定**，收发显示一致（与「HEX 发送」解耦）。
- 修复时间戳偏淡、深色主题下拉白边等观感问题；一批代码审查问题修复与去重。
