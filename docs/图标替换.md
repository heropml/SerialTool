# 图标替换说明

CommTool 的图标分**两层**，换图标时按需改其一或两者：

| 图标 | 显示位置 | 来源 |
|------|----------|------|
| **运行时图标** | 窗口标题栏 / 任务栏 / 系统托盘 / 关于对话框 | `src/icon_data.py` 里的 `ICON_B64`（128×128 PNG 的 base64，**固化在源码内**） |
| **exe 文件图标** | 资源管理器里 exe 文件的图标 | 打包时 PyInstaller `--icon`（正式版用 `assets/icon.ico`） |

> 两者相互独立：运行时图标写死在代码里，别人替换 `icon.ico` 也改不掉窗口/托盘里显示的图标。

---

## 方式 A：临时定制版（推荐，不动正式图标）

给某人打一个「换了头像」的版本、又不想改正式图标时用。**源码零留痕**——脚本会自动还原。

**命令行（最快）：**

```
py -3 scripts\build_custom.py <图片路径> [名字后缀]
```

例：`py -3 scripts\build_custom.py D:\Pictures\avatar.jpg laoba`

**双击（不想敲命令）：**

双击 `scripts\build_custom.bat` → 把图片**拖进黑窗口** → 回车。

约 1 分钟后，产出：

```
dist_onefile\CommTool_onefile_v<版本>_<后缀>.exe
```

（后缀不填默认 `custom`；版本号自动读 `src/version.py`。）

脚本自动完成：图片居中裁方形 → 转 128×128 运行时图标 + 多尺寸 `.ico` → 打包单文件版 → **还原 `src/icon_data.py`、清理临时文件**（即使打包失败或中途中断，下次运行也会先自动还原，绝不污染正式图标）。

> 提示：照片细节多，缩到任务栏 16~32px 会偏糊。想清楚些就用**方形特写**或先把背景抠成透明再做。

---

## 方式 B：正式永久替换（改整个产品的图标）

要把正式版图标永久换掉时用，分三步：

**1. 换 exe 文件图标**（资源管理器里显示的）

- 把新原图存为 `assets\icon_src.png`（或 `.jpg`，任意分辨率，会自动居中裁方形）
- 运行 `python assets\icon_convert.py` 生成 `assets\icon.ico`（自动抠白底 + 边缘羽化）

**2. 换运行时图标**（窗口 / 任务栏 / 托盘）

重新生成 `src/icon_data.py` 的 `ICON_B64`（128×128 PNG 的 base64）：

```
python -c "from PIL import Image; img = Image.open('assets/icon_preview.png').convert('RGBA'); img.thumbnail((128,128), Image.LANCZOS); img.save('_icon_embed.png','PNG',optimize=True)"
python -c "import base64,textwrap; b64='\n'.join(textwrap.wrap(base64.b64encode(open('_icon_embed.png','rb').read()).decode(),76)); open('src/icon_data.py','w',encoding='utf-8').write('# -*- coding: utf-8 -*-\nICON_B64 = \"\"\"\\\n'+b64+'\n\"\"\"\n')"
```

（把 `assets/icon_preview.png` 换成你的图即可。）

**3. 重新打包**

```
scripts\build.bat            文件夹版
scripts\build_installer.bat  安装包
scripts\build_onefile.bat    单文件版
```

---

## 小结

| 需求 | 用 |
|------|----|
| 临时给个换头像的版本 | **方式 A**：`build_custom`，一条命令，自动还原 |
| 永久换掉产品正式图标 | **方式 B**：改 `assets/icon.ico` + 重生成 `icon_data.py` + 重新打包 |
