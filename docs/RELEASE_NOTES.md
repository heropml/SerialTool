串口 + 网络一体调试工具。本版聚焦**自动应答**与**多条发送**的实用增强。

## ✨ 新功能 / 改进

- **自动应答「帧头+长度组帧」** —— 针对有帧头 + 长度字段的协议（如 `AA BB | seq | len | … | sum`），
  维护跨包字节流、按帧头定位 + Length 字段算出真实帧边界切分，正确处理串口的**粘包/拆包**，
  不再靠"整包静默超时"硬等。对话框顶部填帧头、长度字段偏移/宽度/字节序、整帧固定开销即可。
- **多条发送：名称列可拖 + 全选** —— 名称框与数据框之间加可拖分隔条，名称长了拖宽即可看全
  （所有行同步）；顶部新增「全选 / 全不选」（三态，自动反映当前勾选）。
- **窗口最小化 / 最大化** —— 多条发送、关键字高亮对话框补上最小化 / 最大化按钮。
