串口 + 网络一体调试工具。本版为 **bug 修复版**，功能与 1.1.2 完全相同。

## 🐛 修复

- **数据区拖选被新数据"吃掉"** —— 用鼠标拖选了一段文本后，串口/网络新数据到达时，
  原生文本选区会随末尾插入一起延伸，把刚收到的行也"选中"（蓝色）。现在 append 时按
  原偏移钉住用户选区，新数据不再混入。
- **标题栏拖出屏幕顶部后消失、拖不回来** —— 无边框窗口被拖到屏幕最上方（标题栏 y<0）
  后，标题栏移出可视区，所有拖动入口都抓不到，窗口再也拖不下来。现在拖动加屏幕工作区
  边界钳制：标题栏始终留在屏内（顶不出屏、底露整条、左右各留 80px 可点），多显示器也对。
