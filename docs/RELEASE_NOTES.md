# CommTool v1.1.0（收发统计）

串口 + 网络一体调试工具。本版在统一版（v1.0.9：Serial / UDP / UDP 组播 / TCP Server / TCP Client 五种连接合一）基础上，给底部状态栏的收发计数加上**包统计与实时速率**。

## ✨ 本版亮点

- **收发速率 / 包统计** — 底部状态栏 RX/TX 由「纯字节数」升级为 **字节 · 包数 · 实时速率(B/s)**：
  - 包计数（RX = 每到达一块、TX = 每成功发送一次）；
  - 1Hz 采样的实时速率，常驻统计、断开后自然归零；
  - 错误数 >0 时标签追加 ⚠ 标记。
- **悬停看明细** — 鼠标停在 RX/TX 标签上弹出 tooltip：总量 / 包数 / 当前速率 / **峰值速率** / 错误数。
- **右键重置统计** — 在状态栏右键「重置统计」一键清零计数器（不动数据区，跟随语言 / 主题）。
- **macOS tooltip 修复** — 修正 macOS 上 tooltip 背景透明、文字看不清的问题（改用自绘不透明提示框）。
- 三语 i18n 同步，无新依赖、无新文件。其余串口 + 网络合一、10 种校验、关键字高亮、实时日志、9 套主题等功能与 v1.0.9 一致。
