# SerialTool v1.0.5

新增**在线更新**功能。「关于」对话框里一键检查更新、下载、手动安装，不用再手动找新版本。既有串口调试功能保持不变。

## ✨ 新功能

- **在线更新（关于 → 检查更新）** —— 系统托盘右键菜单新增「关于」入口，打开对话框显示版本信息 + 「检查更新」按钮。点一下自动比对最新版本：
  - **有新版** → 显示新版本号 + 更新说明 + 「下载并更新」按钮 → 下载完成后**弹出正常安装向导**，手动点「下一步 / 安装」完成升级（不是静默安装）。
  - **已是最新** → 提示当前已是最新版本。
  - **更新源**：内网优先、自动回退公网；每个源 8 秒超时，外网环境也不会长时间卡在「正在检查…」。
- **一键发版脚本 `scripts/release.ps1`** —— 开发者一条命令完成「改版本号 → 改版本清单 → 打包 → push → 建 Release 传安装包」。

## 🛡️ 健壮性

- 每个更新源 **8 秒超时**，内网连不上快速回退公网（不再卡 ~21 秒）。
- 下载后校验文件头魔数（`MZ`），防止错误页 / 404 被当安装包执行。
- 关闭对话框 / 按 ESC 自动**中止在途的检查 / 下载**，关闭文件句柄、删掉写了一半的安装包。
- 启动时清理上次更新残留在 `%TEMP%` 的安装包。
- 版本号解析容错（`1.0.5-beta` 之类不再抛异常）。
- 基于 Qt 自带 **QtNetwork**，无新增第三方依赖。
