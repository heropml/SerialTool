串口 + 网络一体调试工具。本版在 v1.1.1（波形图 + 帧解析）基础上，重点强化**自动应答**到生产可用、并补齐一系列调试常用功能。

## ✨ 本版亮点

### 🔁 自动应答（v1.1.1 已有，本版做实质性增强）

- **多帧应答**：回复字段用 `|` 分段，每段独立替换占位符 + 独立校验，按延时顺次发（覆盖 ACK + DATA 等多帧响应协议）
- **HEX 通配 `??`**：匹配里 `54 ?? 03` 表示「首字节 54、第 3 字节 03、中间任意」，告别按字段笛卡尔积写多条规则
- **应答占位符大扩展**：
  - `{rN}` `{rN-M}` 回填收到帧字节（基础）
  - `{rN+K}` `{rN^K}` 字节算术（加 K mod 256 / XOR K，K 支持 0xHH）
  - `{seq}` 自增 1 字节计数（每帧 +1，wrap）
  - `{ts}` 当前毫秒时间戳低 4 字节 BE
- **四种时序控制**：
  - **整包超时**：累积字节、静默 N ms 视作整帧再匹配（Modbus 帧间静默分帧）
  - **收包校验**：尾部按设备算法校验通过才应答（MOBUS / CRC16 等 10 种）
  - **应答延时**：匹配后等待 N ms 再回（从机 turnaround）
  - **触发冷却**：同一规则 N ms 内只响一次（防应答风暴）
- **总开关一键切换**：「自动应答」按钮**双击** = 翻转启用状态，启用时按钮高亮
- **使用说明独立窗口**：顶部 `?` 按钮 → 6 个具体例子（含本设备 MOBUS 场景、心跳应答、ACK+DATA 多帧等）

### ⌨️ 发送区生产力升级

- **命令历史**：发送框光标在首行按 ↑ 取上一条、末行按 ↓ 往后翻 / 回到草稿；FIFO 100 条跨会话持久化
- **动态字段**：发送内容里写
  - `{count}` 自增 1 字节计数（发送失败回滚，不消耗）
  - `{ts}` 当前毫秒时间戳 4 字节 BE
  - `{randN}` 随机 N 字节（N=1..256，`{rand}` = `{rand1}`）
- 悬停发送框 1 秒弹出**长寿命 tooltip**（停留期间不超时消失），完整语法 + 实战例子

### 🔌 连接

- **自动重连**：非主动断开（运行时掉线 / 对端关闭 / TCP 异步连接失败）按 1/2/4/8/16/30s 退避自动重试；手动打开失败（端口占用等）不会陷入死循环；用户主动断开 / 退出跳过

### 💾 会话配置档 导入/导出

- **数据区右键菜单**「导出配置… / 导入配置…」（或 Ctrl+Shift+S / Ctrl+Shift+O）
- 一份 JSON 文件包含 35+ 项：连接参数 + 主题/语言 + 多条发送组 + 关键字高亮 + 自动应答规则 + 帧解析规则 + 波形图配置
- 导入后**立即生效**：主题/语言/显示开关/规则全部刷新，已打开的弹窗也同步重建
- 适合：换机器搬配置、协议变种保存多套方案、同事间共享调试预设

### 🔍 其它

- **Ctrl+F 全局快捷键**：任何控件焦点下按 Ctrl+F 都能弹数据区搜索栏并聚焦
- **波形图 + 帧解析**：右上角 `?` 按钮 → 独立窗口 5 个用例（CSV / 正则 / HEX 二进制 / 帧头过滤 / 浮点 IMU）
- **InfoDialog**：导入/导出/错误提示从原生 QMessageBox 换成与主界面同主题的圆角对话框（圆形 ✓/✕ 图标 + accent 蓝按钮）
- 三语 i18n（简中 / English / 繁中）同步本版所有新文案

### 🐛 修复

- 自动应答延时期间关掉总开关 → 排队的回复不会发
- 多条发送循环路径漏过动态字段替换
- 断/重连时清自动应答半包缓冲，避免旧字节被新连接吃
- `{seq}` 仅在 reply 实际用到时才推进（不再空耗）
- 配置导入读 bool 用 `type=bool`（手写 JSON `"false"` 不会被 bool() 误判 True）
- 5 个子对话框 `parent=None` → 修复 "开关任意子弹窗后主窗边缘 resize 失效" 的 Windows 老问题
