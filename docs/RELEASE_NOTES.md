# SerialTool v1.0.7

稳定性与代码审查修复版。串口收发核心功能保持不变。

## 🛡️ 稳定性修复

- **关闭串口更稳健** —— `close_serial` 先断开读取线程（SerialReader）信号再停止，消除串口出错关闭路径下的重复错误提示与重入。
- **读取线程退出余量** —— SerialReader 退出等待时间加大，设备异常时线程不悬挂。

## 🔍 搜索功能修复

- **导航不再卡顿** —— 点 ▲/▼ 上下跳转改为 O(1) 着色，大数据量下不再每次全文重新扫描匹配。
- **打开搜索消除双重扫描** —— 选中文本打开查找栏（Ctrl+F）时不再重复执行两遍全文搜索。
- **实时数据搜索匹配跟随更新** —— 只用搜索、未配置关键字高亮规则时，持续接收的新数据中的匹配现在也会实时高亮、刷新「N/总数」计数并纳入 ▲/▼ 导航。
- 查找栏定位 viewport 守卫、计数标签守卫，规避极端时序下的异常。

## 🔧 其他

- **版本号解析** —— 正确处理预发布后缀（`1.0.7 > 1.0.7-rc1`），不再整段截断。
