# CommTool v1.1.1（数据波形图 + 协议帧解析）

串口 + 网络一体调试工具。本版在 v1.1.0（收发速率/包统计）基础上，新增两大数据分析功能。

## ✨ 本版亮点

- **数据波形图** — 标题栏「波形图」：从 RX 数据解析数值实时绘多通道滚动曲线（示波器式）。
  - 三种解析：**分隔符**（逗号/空白/Tab/分号/自动，每列一条曲线）、**正则**（每个捕获组一条）、
    **HEX 字节字段**（二进制协议用，按 `偏移:类型` 取值 + 帧头过滤）；
  - 通道显隐/配色、窗口点数、X 轴样本序号/时间、暂停/清空/导出 CSV；窗口可最小化/最大化/缩放。
- **协议帧解析表** — 标题栏「帧解析」：把每个接收帧按规则解析成「名称 = 值」表格。
  - **多帧多规则**：规则列表每行「帧头 | 字段定义」，每帧按帧头前缀匹配第一条规则；
  - 「全部」标签看混合帧流 + 每规则一个分列标签；序号列、原始帧列；
  - 字段类型数值 + `hexN`/`strN`，数值加 `x` 后缀按十六进制显示；
  - Ctrl+C / 右键 复制·全选、导出 CSV、暂停、列宽可拖、滚动锁定 +「↓最新」回底。
- 波形图与帧解析共用同一套帧头/字段定义（共享模块 binproto）。三语 i18n 同步。
- 新增依赖 pyqtgraph + numpy（仅波形图用；安装包已内置，无需另装 Python 环境）。

> Windows 10/11（64 位）无需安装 Python。1.0.9/1.1.0 用户「关于 → 检查更新」即可升级。

> **macOS 首次打开提示「已损坏，无法打开」**：不是真的损坏，是 macOS 对未公证下载文件的拦截。把 app 拖进「应用程序」后，终端运行一次 `xattr -dr com.apple.quarantine /Applications/CommTool.app` 即可正常打开（仅支持 Apple Silicon）。
