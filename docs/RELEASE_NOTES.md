串口 + 网络一体调试工具。本版聚焦**两项体验增强**：多条发送支持**拖拽排序**、新增**自动检查更新**（启动 + 定时后台静默查，发现新版不打扰地提示）。

## ✨ 新功能 / 改进

- **多条发送拖拽排序** —— 每行行首新增 **☰ 拖拽手柄**，按住上下拖即可改变发送顺序，松手立即保存；拖到目标行上半 / 下半自动定位插入点。
- **自动检查更新（类 Claude 的静默提示）**：
  - 启动后台静默检查一次，之后**每 6 小时**再查；命中新版才提示，无新版 / 网络不通**全程静默、不弹任何框**。
  - 发现新版时，**右下角版本号变成「● 可更新 vX」高亮可点徽标**（悬停提示“点击查看并更新”），点一下直接打开「关于」走下载安装。
  - 「关于」对话框新增 **「自动检查更新」开关**（默认开）：关掉后停止后台检查、仍可手动点「检查更新」；开关随会话配置档导入导出。
- 评审收尾小修：拖拽手柄按下事件透传给基类保持状态一致、补 `DragLeave` 处理、自动更新空版本号边界防御、主题刷新复用调色板少算一次。
