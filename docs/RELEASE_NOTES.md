# SerialTool v1.0.4

边界鲁棒性修复版。**界面与功能无新增**，修掉三个边界场景下的体验/正确性问题，日常使用更稳。

## 🔧 修复

- **定时发送遇格式错误不再 Toast 轰炸** —— 开了「定时发送」时，若发送内容有问题（如 HEX 格式非法），原先会每个周期都弹一次错误提示、且定时器不停。现在**任何发送失败都会自动关掉定时发送**，只提示一次。

- **断开/出错时不再双重「串口未打开」提示** —— 串口出错或主动关闭（非退出程序路径）时，原先「多条发送循环」定时器不会停，下一拍又会弹一次「串口未打开」。现在 `close_serial` 会**同步停掉循环定时器**，只剩一条准确提示。

- **配置文件落地判定更稳** —— `_settings_file` 探测安装目录是否可写时，原先若测试文件删不掉（杀软锁定等罕见情况）会**误判为不可写**、多余地回退到 `%APPDATA%`，还残留一个空文件。现在**写成功即判定可写**，删测试文件改为 best-effort（删不掉不影响判定、不残留）。
