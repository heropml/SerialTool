# CommTool v1.1.5（自动应答五项增强）

串口 + 网络一体调试工具。本版把**自动应答**从"能用"推到"好用 + 可压测"：校验段、规则测试器、命中统计、故障注入、范围延时。

## ✨ 新功能 / 改进

- **校验段（内层 / 额外校验）** —— 每条应答可加 N 段：对应答帧 `[起始..结束]` 算校验、覆盖到指定偏移或追加，在行尾「校验」之前按序计算。配合 `{rN}` 占位符回显，**内层 CRC 随回显字段自动重算**，解决「外层 Sum + 内层 CRC」只能硬编码的痛点。
- **规则测试器（离线）** —— 顶栏「测试」：输入一帧 HEX → 立即看命中哪条规则 + 应答预览（含占位符替换 + 校验段/尾部校验算好的字节）。与实发共用组装逻辑，**预览=实发**；不发送、不计数、不推进 `{seq}`。
- **命中统计** —— 每条规则行实时显示「命中 N」+ 顶栏「重置统计」；匹配且收包校验通过即计（运行态、不持久化、不导出）。
- **故障注入（全局，压测主机）** —— 按概率 **丢包**（超时不回，测重传）/ **错CRC**（末字节翻转，测校验）/ **错长度**（砍末字节，测分帧）。被注入的帧在数据区有 `⚠ 故障注入·…` 标记。
- **范围延时** —— 应答延时字段支持 `100`（固定）或 `100-300`（每次发随机抖动，模拟设备 turnaround）。
- 「帧头+长度组帧」与「故障注入」框右侧加一句话说明 + `?` 例子帮助窗；三语 i18n 同步；无新依赖。
