串口 + 网络一体调试工具。本版为**稳健性修复版**：串口的选择与掉线处理更可靠，多个对话框拖动调整的列宽现在会记住。

## ✨ 修复 / 改进

- **串口选择防"乱跳"** —— 选中某个串口后，后台扫描时它偶发瞬时掉枚举（USB 转串口常见），不再让选择静默跳到列表里的另一个口；口确实长期不在了，则自动移除「未检测到」占位、回落到第一个真实口（不再常驻残留项）。
- **已连接的串口被拔出 / 掉驱动** —— 连续几次检测不到即**自动断开连接并提示**「串口 X 已移除」，不再卡在"已连接"的假象里。
- **串口掉线不再反复自动重连** —— 串口断开几乎都是拔出 / 断电的物理事件，重连没意义、还可能连到别的口；故串口掉线后干净断开、不再自动重连（网络 TCP / UDP 掉线仍按退避自动重连）。
- **对话框列宽记住了** —— 「多条发送」「Modbus 主机轮询」「自动应答」三个对话框里拖动调整的列宽现在会持久化，重开 / 重启后保持，并随会话配置档导入导出。
- 另修：启动恢复上次串口的时序问题（启动扫描早于配置恢复时仍能选回上次的口）；连接出错处理改用实际连接协议判断，避免连接中导入配置后误判。
