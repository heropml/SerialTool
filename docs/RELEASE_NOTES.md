串口 + 网络一体调试工具。本版新增**「终端模式」**——把发送框变成一个轻量串口终端，适合登录 Linux 串口控制台敲命令调试。

## ✨ 新功能 / 改进

- **终端模式（轻量串口终端）** —— 「发送区」设置里打开「终端模式」开关后：
  - **逐字符即时发送** —— 在发送框直接键入，每个按键立即发给设备：**回车 / Backspace / Tab / Ctrl+C / 方向键（↑↓ shell 历史、←→ 移动）/ ESC / Home / End / Delete** 等都透传。
  - **终端语义渲染** —— 数据区按终端方式显示设备回显：正确处理**退格擦除**、`\r` 回车、光标移动，以及行编辑常用的 ANSI 序列（`ESC[J` / `ESC[K` 擦除、`ESC[C` / `ESC[D` 光标）；**颜色码 `ESC[..m` 自动忽略**（带颜色的 `ls` 等输出显示干净、不再是 `^[[31m` 乱码）。
  - **本地回显 / 回车映射** —— 「本地回显」开关（设备不回显时用）；「回车」可选 **CR / LF / CRLF**（默认 CR，适配多数 Linux 控制台）。
  - 开启终端模式时，**用不上的格式设置（HEX 显示 / 时间戳 / 分包 / 超时 / HEX 发送 / 校验等）自动淡化禁用**、并停掉定时发送，只留终端相关项；关闭后全部恢复。
  - 局限：不解析全屏 TUI（`vi` / `top` 等满屏光标定位），跑那些仍建议用专门终端。
- 终端模式三项随会话配置档导入导出、即时生效；经多轮评审加固（超大 CSI 计数 / 无终止符 CSI 洪水 / 坏配置值等边界）。
