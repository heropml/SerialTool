串口 + 网络一体调试工具。本版在 v1.2.0「Modbus 主机轮询」基础上补齐**写多功能码**、加固**误操作安全**、并把表格做成**可拖拽列**——经六轮对抗式评审收敛。

## ✨ 新功能 / 改进

- **写多功能码** —— 主机写操作补上 **0F 写多线圈 / 10 写多寄存器**：在「数量/写值」格填多个值（逗号或空格分隔，如 `100,200,300`；0F 线圈填 `0/1`），数量由值的个数决定。
- **误操作安全（多轮评审收敛）**：
  - 非法 / 越界的 **从机ID（0..247）、起始地址（0..65535）、写值** 一律**拒发**而非静默钳位 —— 杜绝把非法 ID 变成广播地址 0、误写所有从机。
  - 写多时任一项非法、或超出规范上限（0x10 > 123 / 0x0F > 1968）→ **整组拒发**，不静默截断、不让后续值前移到错误地址。
  - **响应超时静默期**：请求超时后短暂不发下一条，避免迟到响应被误配对成下一行的读取结果。
  - 主机轮询激活时跳过自动应答，防止与「自动应答 · Modbus 从机」互相把响应当请求回发、干扰总线。
- **草稿编辑** —— 启用轮询时编辑规则只进草稿、不再自动提交并发送，须点**「应用」**才生效（防把读改成写时误发一帧）；未应用的草稿关窗重开仍保留。
- **可拖拽列** —— 名称~值各列之间可**左右拖动调宽**，所有行与表头像素同步；功能码列默认更窄。
- 健壮性：单条规则损坏只跳过不清空整表；配置里字符串 `"false"` 正确识别为停用；大量非法规则异步排程，不再递归致栈溢出。`tests/test_modbus_master.py` 增至 60+ 项；无新依赖；关闭 Modbus 主机时行为与 v1.2.0 一致。
- **互斥 + 连接自愈**：自动应答与 Modbus 主机**一键互斥**（开一个自动关另一个、两个对话框开关同步）；TCP/串口半帧写入即时断流重连、不污染后续帧；Modbus 地址/数值输入兼容前导零（如 `08`/`010`）。
