串口 + 网络一体调试工具。本版新增**多窗口**——可同时开多个独立窗口、各调各的设备、互不干扰。

## ✨ 新功能 / 改进

- **多窗口（独立配置）** —— 现在可以同时开多个 CommTool 窗口：
  - **新建窗口** —— 「帮助 → 新建窗口」再开一个独立窗口（也可直接再运行一次程序）；标题带 `(2)` / `(3)` 区分（任务栏、托盘同样带后缀）。
  - **配置互不覆盖** —— 每个窗口用各自的配置文件（主窗口 `settings.ini`，其余 `settings-2.ini` / `settings-3.ini`…），改设置、关窗口不再互相覆盖；靠文件锁自动分配空闲槽位，连**双击开两个**也会自动隔离（解决了以前"开两个窗口配置互相覆盖"的问题）。
  - **各自独立** —— 每个窗口有独立的连接、收发、自动应答、Modbus 主机、终端等状态，互不干扰（注意：同一个串口仍只能被一个窗口打开，第二个开同口会提示占用——这是正常的，数据不会串）。
  - **多显示器兜底** —— 窗口若落到已断开 / 关闭的屏幕外，首次显示时自动搬回主屏，避免"进程在、窗口看不见"。
